The React Google Maps library comes with a comprehensive collection of working examples that demonstrate real-world implementations. All examples are available in the [GitHub repository](https://github.com/visgl/react-google-maps/tree/main/examples) and many have live demos you can interact with.

## Getting Started with Examples

Each example is fully self-contained and can be run independently:

```bash
cd examples/[example-name]
npm install
npm start
```

During development, you can also run examples with the local library source:

```bash
npm run start-local
```

## Example Structure

Each example typically includes:

- `src/app.tsx` - Main application component with the map implementation
- `src/control-panel.tsx` - Optional UI controls for the example
- `vite.config.js` - Build configuration
- `package.json` - Dependencies and scripts

## CodeSandbox Quick Start

When browsing examples on GitHub, you can quickly open any example in CodeSandbox by replacing `github.com` with `githubbox.com` in the URL.

For example:
```
https://github.com/visgl/react-google-maps/tree/main/examples/basic-map
```

Becomes:
```
https://githubbox.com/visgl/react-google-maps/tree/main/examples/basic-map
```
