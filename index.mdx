## Overview

This library provides a collection of React components and hooks to integrate the Google Maps JavaScript API into your React application. Built with TypeScript, it offers full type safety and supports modern React patterns including hooks and server-side rendering.

## Quick Example

```tsx
import {APIProvider, Map, AdvancedMarker} from '@vis.gl/react-google-maps';

function App() {
  const position = {lat: 53.54992, lng: 10.00678};

  return (
    <APIProvider apiKey={'YOUR_API_KEY'}>
      <Map defaultCenter={position} defaultZoom={10} mapId="DEMO_MAP_ID">
        <AdvancedMarker position={position} />
      </Map>
    </APIProvider>
  );
}
```
