This guide will walk you through creating your first interactive map with markers and info windows using React Google Maps.

## Prerequisites

Before starting, make sure you have:

- Installed `@vis.gl/react-google-maps` ([Installation Guide](/installation))
- A valid Google Maps API key
- A Map ID (required for AdvancedMarker components)

## Basic Map Setup

<Steps>
  <Step title="Import Required Components">
    Start by importing the core components from the library:

    ```tsx
    import {APIProvider, Map} from '@vis.gl/react-google-maps';
    ```

    - `APIProvider`: Loads the Google Maps JavaScript API and provides context
    - `Map`: Renders the map component
  </Step>

  <Step title="Create the APIProvider">
    Wrap your application (or map components) with `APIProvider`. This component loads the Google Maps JavaScript API and provides context to all child components.

    ```tsx
    function App() {
      return (
        <APIProvider apiKey={'YOUR_API_KEY_HERE'}>
          {/* Map and other components go here */}
        </APIProvider>
      );
    }
    ```

    <Note>
      The `APIProvider` should typically be placed near the top of your component tree. There should only be one `APIProvider` per application.
    </Note>
  </Step>

  <Step title="Add a Map Component">
    Inside the `APIProvider`, add a `Map` component with camera settings:

    ```tsx
    function App() {
      return (
        <div style={{height: '100vh', width: '100%'}}>
          <APIProvider apiKey={'YOUR_API_KEY_HERE'}>
            <Map
              defaultCenter={{lat: 22.54992, lng: 0}}
              defaultZoom={3}
              gestureHandling='greedy'
              disableDefaultUI
            />
          </APIProvider>
        </div>
      );
    }
    ```

    Key props:
    - `defaultCenter`: Initial map center (latitude/longitude)
    - `defaultZoom`: Initial zoom level (0-22)
    - `gestureHandling`: Controls how gestures interact with the map
    - `disableDefaultUI`: Hides default map controls

    <Warning>
      Maps have an intrinsic height of 0 by default. Always set a height on the map container or parent element.
    </Warning>
  </Step>
</Steps>

## Adding Markers

Now let's add interactive markers to your map:

<Steps>
  <Step title="Import Marker Components">
    Import `AdvancedMarker` for modern, customizable markers:

    ```tsx
    import {APIProvider, Map, AdvancedMarker} from '@vis.gl/react-google-maps';
    ```
  </Step>

  <Step title="Add a Simple Marker">
    Place an `AdvancedMarker` inside the `Map` component:

    ```tsx
    function App() {
      const position = {lat: 53.54992, lng: 10.00678};

      return (
        <APIProvider apiKey={'YOUR_API_KEY_HERE'}>
          <Map 
            defaultCenter={position} 
            defaultZoom={10}
            mapId={'YOUR_MAP_ID'}
          >
            <AdvancedMarker position={position} />
          </Map>
        </APIProvider>
      );
    }
    ```

    <Warning>
      `AdvancedMarker` requires a `mapId` prop on the `Map` component. See the [Installation Guide](/installation#map-ids-for-advanced-markers) for details.
    </Warning>
  </Step>

  <Step title="Customize Markers with Pin">
    Use the `Pin` component to customize marker appearance:

    ```tsx
    import {APIProvider, Map, AdvancedMarker, Pin} from '@vis.gl/react-google-maps';

    function App() {
      return (
        <APIProvider apiKey={'YOUR_API_KEY_HERE'}>
          <Map 
            mapId={'YOUR_MAP_ID'}
            defaultCenter={{lat: 20, lng: 10}} 
            defaultZoom={5}
          >
            <AdvancedMarker position={{lat: 20, lng: 10}}>
              <Pin
                background={'#22ccff'}
                borderColor={'#1e89a1'}
                glyphColor={'#0f677a'}
              />
            </AdvancedMarker>

            {/* Marker with emoji */}
            <AdvancedMarker position={{lat: 15, lng: 20}}>
              <Pin background={'#FBBC04'} scale={1.4}>
                📍
              </Pin>
            </AdvancedMarker>
          </Map>
        </APIProvider>
      );
    }
    ```
  </Step>
</Steps>

## Adding Info Windows

Info windows display content when users interact with markers:

<Steps>
  <Step title="Import InfoWindow">
    Add the `InfoWindow` component to your imports:

    ```tsx
    import {
      APIProvider, 
      Map, 
      AdvancedMarker, 
      InfoWindow,
      useAdvancedMarkerRef
    } from '@vis.gl/react-google-maps';
    ```
  </Step>

  <Step title="Create a Marker with Info Window">
    Combine markers with info windows for interactive content:

    ```tsx
    import {useState} from 'react';
    import {
      AdvancedMarker,
      InfoWindow,
      useAdvancedMarkerRef
    } from '@vis.gl/react-google-maps';

    function MarkerWithInfoWindow() {
      const [infoWindowOpen, setInfoWindowOpen] = useState(false);
      const [markerRef, marker] = useAdvancedMarkerRef();

      return (
        <>
          <AdvancedMarker
            ref={markerRef}
            onClick={() => setInfoWindowOpen(true)}
            position={{lat: 28, lng: -82}}
            title={'Click me!'}
          />
          
          {infoWindowOpen && (
            <InfoWindow
              anchor={marker}
              maxWidth={200}
              onCloseClick={() => setInfoWindowOpen(false)}
            >
              <h2>Location Info</h2>
              <p>This is an AdvancedMarker with an InfoWindow!</p>
            </InfoWindow>
          )}
        </>
      );
    }
    ```

    Key concepts:
    - `useAdvancedMarkerRef`: Hook to get a reference to the marker
    - `anchor`: Links the info window to the marker
    - `onCloseClick`: Handler for the close button
  </Step>
</Steps>

## Complete Example

Here's a complete working example with multiple markers and info windows:

```tsx
import React, {useState} from 'react';
import {createRoot} from 'react-dom/client';
import {
  APIProvider,
  Map,
  AdvancedMarker,
  Pin,
  InfoWindow,
  useAdvancedMarkerRef
} from '@vis.gl/react-google-maps';

// Marker with InfoWindow component
function MarkerWithInfoWindow({position, title, children}) {
  const [infoWindowOpen, setInfoWindowOpen] = useState(false);
  const [markerRef, marker] = useAdvancedMarkerRef();

  return (
    <>
      <AdvancedMarker
        ref={markerRef}
        onClick={() => setInfoWindowOpen(true)}
        position={position}
        title={title}
      />
      {infoWindowOpen && (
        <InfoWindow
          anchor={marker}
          maxWidth={200}
          onCloseClick={() => setInfoWindowOpen(false)}
        >
          {children}
        </InfoWindow>
      )}
    </>
  );
}

// Main App component
function App() {
  const locations = [
    {lat: 53.54992, lng: 10.00678, name: 'Hamburg'},
    {lat: 52.52437, lng: 13.41053, name: 'Berlin'},
    {lat: 48.13743, lng: 11.57549, name: 'Munich'}
  ];

  return (
    <div style={{height: '100vh', width: '100%'}}>
      <APIProvider apiKey={process.env.GOOGLE_MAPS_API_KEY}>
        <Map
          mapId={'bf51a910020fa25a'}
          defaultCenter={{lat: 51.5, lng: 10.5}}
          defaultZoom={6}
          gestureHandling={'greedy'}
        >
          {locations.map((location, index) => (
            <MarkerWithInfoWindow
              key={index}
              position={{lat: location.lat, lng: location.lng}}
              title={location.name}
            >
              <div>
                <h3 style={{margin: 0}}>{location.name}</h3>
                <p>Latitude: {location.lat}</p>
                <p>Longitude: {location.lng}</p>
              </div>
            </MarkerWithInfoWindow>
          ))}

          {/* Custom styled marker */}
          <AdvancedMarker position={{lat: 50.1109, lng: 8.6821}}>
            <Pin
              background={'#FBBC04'}
              borderColor={'#F9AB00'}
              glyphColor={'#F57C00'}
            />
          </AdvancedMarker>
        </Map>
      </APIProvider>
    </div>
  );
}

export default App;

// Render to DOM
const root = createRoot(document.getElementById('root'));
root.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);
```

## Using Other Google Maps Libraries

The Maps JavaScript API provides additional libraries for geocoding, routing, places, and more. Load them dynamically with the `useMapsLibrary` hook:

```tsx
import {useMapsLibrary} from '@vis.gl/react-google-maps';
import {useEffect, useMemo} from 'react';

function GeocodingComponent() {
  // Load the geocoding library
  const geocodingLib = useMapsLibrary('geocoding');
  
  // Create geocoder instance
  const geocoder = useMemo(
    () => geocodingLib && new geocodingLib.Geocoder(),
    [geocodingLib]
  );

  useEffect(() => {
    if (!geocoder) return;

    // Use the geocoder
    geocoder.geocode(
      {address: '1600 Amphitheatre Parkway, Mountain View, CA'},
      (results, status) => {
        if (status === 'OK' && results[0]) {
          console.log(results[0].geometry.location);
        }
      }
    );
  }, [geocoder]);

  return null;
}
```

<Note>
  `useMapsLibrary` returns `null` initially while the library loads, then returns the library object once ready.
</Note>

## Key Concepts

### Controlled vs Uncontrolled

Maps can be controlled or uncontrolled:

**Uncontrolled** (default): Use `defaultCenter` and `defaultZoom` - the map manages its own state

```tsx
<Map defaultCenter={{lat: 40, lng: -74}} defaultZoom={12} />
```

**Controlled**: Use `center` and `zoom` with state - you manage the map state

```tsx
function ControlledMap() {
  const [center, setCenter] = useState({lat: 40, lng: -74});
  const [zoom, setZoom] = useState(12);

  return (
    <Map 
      center={center} 
      zoom={zoom}
      onCameraChanged={(ev) => {
        setCenter(ev.detail.center);
        setZoom(ev.detail.zoom);
      }}
    />
  );
}
```

### Styling the Map

Style the map container using `style` or `className` props:

```tsx
<Map
  style={{width: '100%', height: '600px', borderRadius: '8px'}}
  className="custom-map"
  defaultCenter={{lat: 40, lng: -74}}
  defaultZoom={10}
/>
```
