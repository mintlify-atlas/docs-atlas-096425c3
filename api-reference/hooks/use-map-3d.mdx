---
title: 'useMap3D'
description: 'Access 3D map instances for photorealistic Google Maps experiences'
---

The `useMap3D()` hook provides access to [`google.maps.maps3d.Map3DElement`](https://developers.google.com/maps/documentation/javascript/reference/3d-map#Map3DElement) instances created by a `Map3D` component within the `APIProvider` in your application.

## Parameters

<ParamField path="id" type="string" optional>
  The id of the map3d instance to retrieve. If not specified, returns the parent map3d instance or the default map3d instance.
</ParamField>

## Returns

<ResponseField name="map3d" type="google.maps.maps3d.Map3DElement | null">
  The 3D map instance, or `null` if it can't be found.
  
  The returned map3d instance is determined by priority:
  1. If an `id` is specified, the map with that `id` is retrieved from the `APIProviderContext`
  2. If there is a parent map3d instance (from context), return it
  3. Otherwise, return the map3d with id `default`
</ResponseField>

## Usage

### Single 3D Map Instance

When there is only a single map within the `APIProvider`, the `useMap3D()` hook can be called without any arguments.

```tsx
import { useMap3D } from '@vis.gl/react-google-maps';
import { useEffect } from 'react';

const MyComponent = () => {
  const map3d = useMap3D();

  useEffect(() => {
    if (!map3d) return;

    // Access 3D map properties
    console.log('Map center:', map3d.center);
    console.log('Camera range:', map3d.range);
  }, [map3d]);

  return <>...</>;
};

const App = () => {
  return (
    <APIProvider apiKey="YOUR_API_KEY">
      <Map3D
        defaultCenter={{lat: 37.7749, lng: -122.4194, altitude: 500}}
        defaultRange={2000}
      >
        <MyComponent />
      </Map3D>
    </APIProvider>
  );
};
```

### Multiple 3D Map Instances

When there are multiple `Map3D` components, use explicit IDs to access specific instances:

```tsx
const MyComponent = () => {
  const mainMap = useMap3D('main-map');
  const miniMap = useMap3D('mini-map');

  useEffect(() => {
    if (!mainMap || !miniMap) return;

    // Synchronize both map instances
    mainMap.addEventListener('gmp-centerchange', () => {
      miniMap.center = mainMap.center;
    });
  }, [mainMap, miniMap]);

  return <>...</>;
};

const App = () => {
  return (
    <APIProvider apiKey="YOUR_API_KEY">
      <Map3D
        id="main-map"
        defaultCenter={{lat: 37.7749, lng: -122.4194, altitude: 500}}
        defaultRange={2000}
      />
      <Map3D
        id="mini-map"
        defaultCenter={{lat: 40.7128, lng: -74.006, altitude: 500}}
        defaultRange={5000}
      />
      <MyComponent />
    </APIProvider>
  );
};
```

## Common Patterns

### Camera Animations

Trigger camera animations using the 3D map instance:

```tsx
const CameraControls = () => {
  const map3d = useMap3D();

  const handleFlyAround = () => {
    if (!map3d) return;

    // Cast to access animation methods
    const map3dWithAnimations = map3d as google.maps.maps3d.Map3DElement & {
      flyCameraAround: (
        options: google.maps.maps3d.FlyAroundAnimationOptions
      ) => void;
      flyCameraTo: (options: google.maps.maps3d.FlyToAnimationOptions) => void;
      stopCameraAnimation: () => void;
    };

    map3dWithAnimations.flyCameraAround({
      camera: {
        center: {lat: 37.7749, lng: -122.4194, altitude: 0},
        range: 1000,
        heading: 0,
        tilt: 60
      },
      durationMillis: 10000,
      repeatCount: 1
    });
  };

  return <button onClick={handleFlyAround}>Fly Around</button>;
};
```

<Tip>
  For camera animations, consider using the `Map3DRef` exposed via `useRef` on the `Map3D` component, which provides typed access to animation methods without casting.
</Tip>

### Monitoring Camera State

Listen to camera changes and react to position updates:

```tsx
const CameraInfo = () => {
  const map3d = useMap3D();
  const [cameraState, setCameraState] = useState<string>('');

  useEffect(() => {
    if (!map3d) return;

    const updateState = () => {
      setCameraState(
        `Center: ${JSON.stringify(map3d.center)}, ` +
          `Range: ${map3d.range}, ` +
          `Heading: ${map3d.heading}`
      );
    };

    map3d.addEventListener('gmp-centerchange', updateState);
    updateState();

    return () => {
      map3d.removeEventListener('gmp-centerchange', updateState);
    };
  }, [map3d]);

  return <div>{cameraState}</div>;
};
```

### Adjusting Camera Properties

```tsx
const CameraController = () => {
  const map3d = useMap3D();

  const adjustCamera = () => {
    if (!map3d) return;

    // Modify camera properties
    map3d.center = {lat: 40.7128, lng: -74.006, altitude: 0};
    map3d.range = 1500;
    map3d.heading = 90;
    map3d.tilt = 45;
  };

  const resetCamera = () => {
    if (!map3d) return;

    map3d.tilt = 0;
    map3d.heading = 0;
  };

  return (
    <div>
      <button onClick={adjustCamera}>Adjust Camera</button>
      <button onClick={resetCamera}>Reset View</button>
    </div>
  );
};
```

<Note>
  The `useMap3D()` hook must be used within a component tree wrapped by an `APIProvider`. If the context cannot be retrieved, an error will be logged and `null` will be returned.
</Note>
